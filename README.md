# Elgato Stream Deck Profile for Mac

## Demo
![alt text](https://github.com/Hakky54/stream-deck-profile/blob/master/images/demo.gif?raw=true)

## Profile content
- IntelliJ Shortcuts
- Spotify shortcuts
- Microsoft Teams shortcuts
- CPU usage
- Date and time
- Media keys
- Pasting unformatted text
- Opening my workspace apps
  - Intelijj IDEA
  - Microsoft Outlook
  - Microsoft Teams
  - Slack
  - Spotify
